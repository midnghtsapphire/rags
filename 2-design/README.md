# Phase 2: Design - Visual & Technical Design

**Purpose:** Create visual designs and technical architecture before coding  
**Duration:** 2-6 weeks  
**Cost:** $5,000-20,000  
**When:** After planning, before development

---

## Overview

Design phase transforms plans into concrete visual mockups and technical specifications. This phase prevents costly rework during development.

---

## Subphases

### 1. Wireframes
Low-fidelity sketches of UI layout

**Tools:** Figma, Balsamiq, Sketch  
**Deliverables:** Wireframe files for all key screens  
**Cost:** $500-2,000

### 2. Mockups
High-fidelity visual designs with colors, fonts, images

**Tools:** Figma, Adobe XD, Sketch  
**Deliverables:** Pixel-perfect mockups for all screens  
**Cost:** $2,000-8,000

### 3. Prototypes
Interactive clickable demos

**Tools:** Figma, InVision, Framer  
**Deliverables:** Clickable prototype for user testing  
**Cost:** $1,000-4,000

### 4. User Flows
Diagrams showing how users navigate

**Tools:** Miro, Lucidchart, Whimsical  
**Deliverables:** User flow diagrams  
**Cost:** $500-2,000

### 5. UML Diagrams
Technical class diagrams, sequence diagrams

**Tools:** Rational Rose, PlantUML, Lucidchart  
**Deliverables:** UML documentation  
**Cost:** $1,000-3,000

### 6. ERD (Database Design)
Entity relationship diagrams

**Tools:** dbdiagram.io, ERwin, MySQL Workbench  
**Deliverables:** Database schema  
**Cost:** $500-2,000

### 7. Architecture Diagrams
System design and data flow

**Tools:** draw.io, Lucidchart, Miro  
**Deliverables:** Architecture documentation  
**Cost:** $1,000-3,000
