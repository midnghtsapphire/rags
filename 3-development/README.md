# Phase 3: Development - Build the Product

**Purpose:** Write code and build the product  
**Duration:** 3-6 months  
**Cost:** $20,000-100,000  
**When:** After design is complete

---

## Project Management Approaches

### Kanban
**Use when:** Continuous flow, no fixed sprints, small team  
**Folder:** `/3-development/kanban/`  
**Tools:** Trello, GitHub Projects

### Scrum
**Use when:** Fixed 2-week sprints, larger team, predictable velocity  
**Folder:** `/3-development/scrum/`  
**Tools:** Jira, Azure DevOps

### Agile
**Use when:** Fast iteration, changing requirements, startup environment  
**Folder:** `/3-development/sprints/`  
**Tools:** Linear, Jira, Monday.com
